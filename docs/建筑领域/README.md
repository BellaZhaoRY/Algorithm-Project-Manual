# 建筑领域相关概念


* [单项工程、单位工程、分部工程、分项工程的划分](https://zhuanlan.zhihu.com/p/349465948)
    * 简单定义
        * 3、分部工程是指———按专业性质、建筑部位确定的；
        * 4、子分部工程是指——当分部工程较大或较复杂时，可按材料种类、施工特点、施工程序、专业系统及类别等划分为若干子分部工程。
        * 5、分项工程是指———按主要工种、材料、施工工艺、设备类别等进行划分；
    * 具体定义
        * 3、单位工程：指竣工后不可以独立发挥生产能力或效益，但具有独立设计，能够独立组织施工的工程。例：土建、电器照明、给水排水等。
        * 4、分部工程：按照工程部位、设备种类和型号、使用材料的不同划分。例：基础工程、砖石工程、混凝土及钢筋混凝土工程、装修工程、屋面工程等。
        * 5、分项工程：按照不同的施工方法、不同的材料、不同的规格划分。例：砖石工程可分为砖砌体、毛石砌体两类，其中砖砌体可按部位不同分为内墙、外墙、女儿墙。分项工程是计算工、料及资金消耗的最基本的构造要素。


* 专业工程,分部工程,分项工程具体含义参考

|分类|概念|定义|
|---|---|---|
|**专业**|土建专业|土建专业的工程包括建筑物的地基、地下室、框架结构、墙体结构、屋面结构、外墙保温等方面的设计和施工|
|*分部*|地基与基础|地基与基础是建筑工程中的分部工程之一。地基与基础分部工程主要包括建筑物地基的处理和基础结构的设计和施工。地基处理一般包括填筑、挖土、加固、灌浆、预应力锚固、地基降渗等工程。基础结构一般包括基础底座、基础柱、基础墙等部分|
|*分部*|主体工程|主要包括建筑物的结构工程和施工工程。结构工程是指建筑物的骨架结构，包括建筑物的柱、梁、板、墙等主要构件以及连接它们的节点和构造。施工工程是指建筑物的施工过程，包括施工方案、施工工艺、施工组织、施工设备和施工人员的管理等|
|*分部*|二次结构工程|它是主体结构工程之后的一部分，二次结构工程主要包括建筑物的外墙、屋面、楼板、楼梯、扶手、窗户、门等非承重结构部分。|
|*分部*|保温工程|保温工程主要包括建筑物的外墙保温、屋面保温、地面保温、窗户保温等部分。|
|*分部*|屋面工程|主要是指建筑物的屋面结构和屋面装饰面的设计和施工。屋面结构包括屋面的承重结构和防水层结构。承重结构一般采用钢筋混凝土构造或钢结构构造，以承受荷载并保证屋面的稳定性和耐久性。防水层结构则包括防水层材料、防水层层厚、防水层施工工艺等内容，以确保屋面的防水性能。屋面装饰面包括屋面的保温层、隔热层、防火层、排水层等。保温层用于保温隔热，隔热层用于隔热保温，防火层用于保障屋面在火灾时的安全性，排水层用于排水和防潮。|
|*分部*|防水工程|防水工程主要包括建筑物的地下室、厨房、卫生间、露台、屋顶等部分的防水设计和施工|
